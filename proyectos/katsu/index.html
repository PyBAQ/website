Katsu

Programa para rifar camisetas, libros o cualquier elemento que se desee regalar durante un evento.

PyBAQ/katsu

Motivación

Poder usar python para hacer algo divertido, por ejemplo rifando cosas en los eventos

Objetivo general

Rifar artículos entre los asistentes a un evento de manera programática

Objetivos específicos

  • Permitir la entrada masiva del registro de asistentes a una charla
  • Escoger entre los asistentes de una charla quien se gana un premio particular
