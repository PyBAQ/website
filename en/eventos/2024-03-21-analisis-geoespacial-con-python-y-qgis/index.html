Análisis Geoespacial con Python y QGIS

Esta charla busca aprovechar el runtime de Python dentro de QGIS para desarrollar algoritmos que automaticen procesos utilizados generalmente en el contexto de emergencias y desastres. Mediante cortos scripts, se mostrarán como manejar los datos geoespaciales para obtener un resultado y la integración de estos dentro de procesos más complejos.
